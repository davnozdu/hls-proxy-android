# HLS Proxy для Android (TV / телефон)

APK-обёртка, которая запускает локальный сервис **[HLS Proxy](https://www.hls-proxy.com/) 8.4.8** прямо на Android-устройстве — без Termux, root и ручной настройки. Предназначена для буферизации IPTV-потоков на приставке.

Делает то же, что инструкция для Termux, но в виде обычного приложения:
- запускает встроенный бинарник `hls-proxy` в фоне (foreground-сервис + wake-lock);
- автозапуск после включения устройства;
- интерфейс с пультом (Android TV) и тачем (телефон/планшет).

## Возможности

- **Пуск / Стоп / Перезапуск** прокси.
- **Статус**: «Работает / Остановлен» и текущий адрес `http://IP:порт`.
- **Смена порта** (по умолчанию `9393`).
- **Автозапуск** при загрузке (переключатель).
- Просмотр журнала работы.

Веб-интерфейс самого HLS Proxy (плейлисты, EPG, настройки) открывается в браузере по адресу из статуса: `http://IP_УСТРОЙСТВА:порт`. Плейлист отдаётся по `http://IP_УСТРОЙСТВА:порт/playlist.m3u8`.

## Технические детали

- Встраивается только сборка **`linux-arm64`** (64-битные Android TV приставки / телефоны, напр. Amlogic S905X5M).
- Бинарник статический (musl, Node.js 16) — исполняется на Android нативно, без proot/chroot. DNS работает напрямую.
- Бинарник кладётся в `jniLibs/arm64-v8a/libhlsproxy.so`, чтобы попасть в `nativeLibraryDir` — единственный каталог, откуда Android разрешает запуск исполняемых файлов.
- Рабочие файлы (`local.json`, кэш) хранятся в `filesDir/hls`; бинарник запускается с этой рабочей папкой и аргументами `-address 0.0.0.0 -port <порт>`.

## Сборка

Сборка идёт на **GitHub Actions** (`.github/workflows/build.yml`). Бинарник и конфиги скачиваются с `hls-proxy.com` на этапе сборки (с проверкой SHA-256) и встраиваются в APK — в репозитории их нет.

- Любой push в `main` или запуск workflow вручную → APK в артефактах сборки (`hls-proxy-apk`).
- Push тега `vX.Y` → APK дополнительно публикуется в Release.

Готовый `app-debug.apk` подписан debug-ключом и ставится сайдлоадом (никаких секретов/keystore не требуется).

### Локальная сборка (опционально)

Нужен JDK 17 и Android SDK. Перед сборкой положить файлы вручную:

```
app/src/main/jniLibs/arm64-v8a/libhlsproxy.so   # бинарник hls-proxy (linux-arm64)
app/src/main/assets/hls/default.json
app/src/main/assets/hls/groups.json
app/src/main/assets/hls/favicon.png
app/src/main/assets/hls/plugins/...
```

Затем: `gradle assembleDebug` (APK в `app/build/outputs/apk/debug/`).

## Установка на приставку

1. Скачать APK из Release или артефактов сборки.
2. Включить установку из неизвестных источников.
3. Установить (через файловый менеджер или `adb install`).
4. Открыть приложение → **Пуск**. Разрешить автозапуск/работу в фоне при запросе.
5. На другом устройстве открыть `http://IP_ПРИСТАВКИ:9393` и настроить плейлисты.
